# 结构生力理论（Structure Force Theory）
一种可计算、可建模、跨尺度统一的宇宙演化理论。

## 核心理念
- 结构（Structure）：一切系统的存在形式
- 内旋（Inner Rotation）：历史、记忆、惯性
- 外扰（External Disturbance）：外部冲击、变化
- 生力（Structure Force）：内旋与外扰的张力驱动系统稳定、进化、崩解

## 嵌套规则
下层结构 = 上层的内旋
上层结构 = 下层的外扰
一套规律，从量子到宇宙全尺度适用。
